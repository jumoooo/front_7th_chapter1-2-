## ✅ 완료된 수용 기준 (Acceptance Criteria)

### AC 1: 반복 유형 선택 기능 ✅

- 일정 생성/수정 폼에 반복 체크박스 구현
- 반복 유형 드롭다운 (매일/매주/매월/매년) 구현
- 반복 간격 입력 필드 구현 (1 이상 정수)
- 반복 종료일 입력 필드 구현 (최대 2025-12-31)
- 유효성 검사 구현

### AC 2: 반복 날짜 계산 로직 ✅

- **매일 반복**: 시작일~종료일, 지정 간격 (4개 테스트 통과)
- **매주 반복**: 시작일~종료일, 지정 주 간격 (3개 테스트 통과)
- **매월 반복**:
  - 일반 케이스: 동일한 날짜 생성 (2개 테스트 통과)
  - 31일 특수 케이스: **31일이 없는 달은 건너뛰기** (3개 테스트 통과)
    - ✅ 예: 1/31 → 3/31 → 5/31 (2월, 4월, 6월, 9월, 11월 건너뜀)
- **매년 반복**:
  - 일반 케이스: 동일한 날짜 생성 (1개 테스트 통과)
  - 윤년 2/29 특수 케이스: **윤년이 아닌 해는 건너뛰기** (3개 테스트 통과)
    - ✅ 예: 2024-02-29 → 2028-02-29 (2025, 2026, 2027년 건너뜀)
- 종료일 초과 방지 (2개 테스트 통과)

### AC 6: 반복 일정 겹침 처리 ✅

- 반복 일정은 일정 겹침을 고려하지 않음 (명세대로 구현)
